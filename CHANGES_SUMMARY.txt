🔐 SECURITY FEATURES IMPLEMENTED
================================================================================

1. JWT Token Management
   • Access tokens: 30 minutes (short-lived)
   • Refresh tokens: 7 days (long-lived)
   • Token rotation: New tokens on each refresh
   • Token blacklisting: Revoked tokens invalid

2. Automatic Logout on Tab Close
   • beforeunload event handler
   • navigator.sendBeacon() for reliability
   • Session marked as revoked
   • localStorage cleared immediately

3. Session Tracking
   • Device fingerprinting (browser/OS hash)
   • IP address logging
   • Device name detection
   • Activity timestamps
   • Multi-device support

4. Role-Based Access Control
   • EMPLOYEE: menu browsing, own orders only
   • ADMIN: full system access
   • Enforced via JWT claims + permission classes

5. Periodic Session Validation
   • Validates every 5 minutes
   • Checks session status, expiry, revocation
   • Auto-logout if invalid

6. Password & Data Security
   • PBKDF2 hashing with salt
   • Passwords never in API responses
   • Serializer field filtering
   • Sensitive fields protected
